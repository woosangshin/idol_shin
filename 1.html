    웹브라우저와 웹서버 관계

    하나는 웹브라우저, 하나는 웹서버. 각각 컴퓨터에 존재해야 인터넷을 통해 연결되는 것이다.
    웹브라우저에서 보고싶은 페이지의 정보를 보내면 웹서버에서 원하는 페이지를 꺼내서 보내준다. 마치 도서관에서 책을 빌려 보여주듯이. 웹브라우저는 request 하고 웹 서버는 response 하는 것이다.

    이것의 관계를 client 와 server 라고 정의 한것이다.


    웹서버를 직접 컴퓨터에 까는 방법과 직접 대행해주는 방법이 있다.
    맡기는것은 웹포스팅.
    웹포스팅을 대신 해주는 회사는 웹호스팅이라고 불리운다.
    웹서버를 제공해주는 회사이다.
    Github 라는 서비스가 무료로 해준다.
